# MaquinaEnigma
Recreación de la maquina Enigma con tres rotores configurables y menú interactivo. 
Permite cifrar y descifrar mensajes de forma reversible siguiendo el funcionamiento histórico.

Características principales:
-Tres rotores configurables: el usuario puede definir la posición inicial de cada rotor.
-Menú interactivo: interfaz sencilla para introducir mensajes y configurar parámetros.
-Cifrado y descifrado reversible: cualquier mensaje cifrado puede ser descifrado con la misma configuración.
-Simulación histórica: se respeta el principio de funcionamiento de la máquina original.

Instalación y uso:
1.Clonar el repositorio: https://github.com/CarmenMaruri04/MaquinaEnigma.git
2.Abrir GitKraken y crear una cópia del repositorio.
3.Ir a VisualStudio y abrir el trabajo en la carpeta.
